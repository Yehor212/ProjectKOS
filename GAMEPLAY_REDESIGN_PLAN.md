# GAMEPLAY REDESIGN PLAN — ProjectKOS v2.0

> Синтез анализа 3 агентов (Gameplay Architect + Pedagogy Curator + Child Logic Auditor)
> Дата: 2026-03-26. Мандат: ПОЛНЫЙ карт-бланш на переписку.

## Принципы редизайна

1. **Каждая игра = персонаж с потребностью + ребёнок помогает**
2. **Каждое действие -> видимая реакция < 100ms**
3. **Ascending reward: tap_stars -> sparkle -> golden_burst -> rainbow**
4. **Difficulty = качественное изменение, не +1 элемент**
5. **Content pool >= 5 сессий без повтора**
6. **Нарратив визуальный, не текстовый**

## 4 системных изменения (cross-cutting)

1. **CharacterReactor** — компонент: животное наблюдает + реагирует на каждое действие
2. **NarrativeIntro** — 3-секундная визуальная заставка с персонажем + потребностью
3. **ContentPoolExpander** — процедурная генерация вариаций где возможно
4. **RewardEscalator** — единая система combo -> ascending pitch -> celebration tiers
