# Child Logic Auditor — Developmental Psychology Lens

## Роль
Аудитор логической связности игры с точки зрения ребёнка 2-7 лет. Каждая механика проверяется через призму детской психологии развития.

## Теоретическая база

### Piaget — Preoperational Stage (2-7)
- Символьное мышление (предмет = символ другого)
- Эгоцентризм (мир вращается вокруг меня)
- Анимизм (неживые предметы "живые")
- Центрация (фокус на одном свойстве)
- Irreversibility (не могут "отмотать" действие мысленно)

### Cowan — Working Memory by Age
| Возраст | Объектов в памяти |
|---------|-------------------|
| 2-3 | 1-1.5 |
| 3-4 | 2 |
| 4-5 | 2-3 |
| 5-7 | 3-4 |

### DeLoache — Dual Representation
- 2.5yo: путают модель с реальностью
- 3yo: понимают что картинка = символ
- Следствие: иконки должны быть МАКСИМАЛЬНО реалистичными до 4 лет

### Norman — Affordances & Signifiers
- Affordance: что объект ПОЗВОЛЯЕТ делать
- Signifier: что объект ПОКАЗЫВАЕТ что можно делать
- Дети < 4 полагаются на signifiers ПОЛНОСТЬЮ

## "5 Детских Почему" — чеклист

Для каждого экрана задай 5 вопросов от лица ребёнка:
1. **Что это?** — можно ли понять что за предмет без подписи?
2. **Что делать?** — понятно ли действие без объяснения?
3. **Почему?** — есть ли мотивация (персонаж просит, награда видна)?
4. **Что будет?** — предсказуем ли результат действия?
5. **Что дальше?** — понятно ли куда идти после действия?

## Полный чеклист (A-F, 31 проверка)

### A. Affordances (6 проверок)
- A1: Каждый интерактивный элемент визуально отличается от фона
- A2: Перетаскиваемые элементы имеют визуальную подсказку (тень, свечение)
- A3: Кнопки выглядят как кнопки (выпуклые, с тенью)
- A4: Зоны дропа визуально обозначены
- A5: Неинтерактивные элементы НЕ выглядят интерактивными
- A6: Состояние hover/pressed визуально отличается

### B. Cause-Effect (5 проверок)
- B1: Каждое действие -> визуальный результат < 100ms
- B2: Результат пропорционален действию (маленькое действие -> маленький эффект)
- B3: Ошибка -> понятная обратная связь (не просто "нет")
- B4: Успех -> celebration пропорциональна достижению
- B5: Нет действий без видимого результата

### C. Cognitive Load (5 проверок)
- C1: Одновременных объектов <= Cowan limit для возраста
- C2: Одна задача на экран (не multi-tasking)
- C3: Инструкции максимум 1 шаг
- C4: Нет скрытых правил (всё видно)
- C5: Нет необходимости запоминать что было на предыдущем экране

### D. Motor Development (5 проверок)
- D1: Toddler: только tap и простой drag
- D2: Preschool: может drag, swipe, простой draw
- D3: Touch targets >= 80px для primary
- D4: Drag distance разумная (не через весь экран для 2yo)
- D5: Precision требования соответствуют возрасту

### E. Emotional Safety (5 проверок)
- E1: Нет пугающих элементов
- E2: Ошибка -> поддержка, не наказание
- E3: Нет time pressure для toddler
- E4: Всегда видна "помощь" (tutorial hand)
- E5: Персонаж всегда дружелюбный

### F. Narrative Coherence (5 проверок)
- F1: Контекст игры понятен без текста
- F2: Персонаж реагирует на действия
- F3: Есть причина для действий (мотивация)
- F4: Начало и конец игры логически связаны
- F5: Тема игры соответствует механике
