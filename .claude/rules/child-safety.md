# Child Safety Rules (Ages 2-7)

1. Always use gentle feedback for Toddler errors: wobble + "click" sound
2. Always count errors for Preschool: `_errors += 1` + "error" sound + vibration
3. Always show tutorial hand on first play (A1: zero-text onboarding)
4. Always scaffold: Toddler 2 errors -> show answer, Preschool 3 errors
5. Always keep touch targets >= 80px for primary buttons
6. Always limit choices to 3-4 per screen (Hick's Law)
7. Always ensure game is playable with sound OFF
8. Always keep animations calm: no flashing > 3Hz, max 100 particles
9. Always protect parental zone with 3-finger gate (LAW 27)
10. Always use COPPA-safe design: no data collection, no tracking
