# Internationalization Rules (4 Languages)

1. Always use `tr("KEY")` for all visible text
2. Always add new keys to `translations.csv` with all 4 columns: en, uk, fr, es
3. Always keep text separate from sprites (no baked-in text)
4. Always use minimum 24px font size for readability
5. Always test that labels fit in all languages (Ukrainian/French tend to be longer)
6. Always use Nunito font (supports Latin + Cyrillic)
7. Always verify translation keys exist before referencing them
