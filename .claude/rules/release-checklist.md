# Release Checklist Rules

1. Always verify export presets are valid (arm64-v8a, com.kosgames.animalpuzzle)
2. Always keep APK < 100MB
3. Always verify privacy policy is current (COPPA 2026 compliant)
4. Always update store listing screenshots for new features
5. Always bump version in project.godot before release
6. Always guard debug prints with OS.is_debug_build()
7. Always exclude tests/ from export
8. Always audit GAME_DESIGN_BIBLE.md is current
9. Always verify translations.csv is complete (4 languages, no empty cells)
10. Always verify Data Protection Program document exists
